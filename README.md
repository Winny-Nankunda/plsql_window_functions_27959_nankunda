1. Problem Definition
Analyzes online store data to understand customer behavior, product demand, and sales trends.
Aims to improve business decisions, increase sales, and enhance customer satisfaction.

2. Success Criteria
Uses rankings, growth comparisons, segmentation, and moving averages to evaluate performance.
These metrics reveal trends and support strategic planning.

3. Database Schema Design
Organizes data into Customers, Products, and Orders with clear key relationships.
Enables accurate tracking of purchases, users, and product performance.

4. Analytical Queries Implemented
Applies SQL window functions to segment customers and analyze sales patterns.
Provides clearer insights into spending behavior and long-term trends.

5. Project Outcome
Identifies top products, valuable customers, and overall sales performance.
Supports data-driven marketing, inventory, and growth decisions.

6. Technologies Used
Relies on SQL, relational databases, and data analytics methods.
These tools enable structured storage, querying, and insight generation.
